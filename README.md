# Desafio-Dois-Calculadora-de-Partidas-Rankeadas
🧮 Calculadora de Partidas Rankeadas

Projeto desenvolvido como parte de um desafio de lógica de programação da plataforma Digital Innovation One.

O objetivo do desafio é aplicar conceitos básicos de programação utilizando JavaScript para calcular o saldo de partidas ranqueadas de um jogador e determinar seu nível.

🎯 Objetivo

Criar uma função que receba como parâmetro a quantidade de vitórias e derrotas de um jogador.

O programa deve calcular o saldo de vitórias utilizando a fórmula:

saldo = vitórias - derrotas

Depois disso, o jogador será classificado em um nível de ranking com base na quantidade de vitórias.

🏆 Sistema de Ranking
Vitórias	Nível
≤ 10	Ferro
11 – 20	Bronze
21 – 50	Prata
51 – 80	Ouro
81 – 90	Diamante
91 – 100	Lendário
≥ 101	Imortal
💻 Conceitos Utilizados

Os conceitos aplicados neste projeto foram:

Variáveis

Operadores aritméticos

Funções

Estruturas condicionais (if / else)

Template strings

📤 Saída Esperada

Ao final da execução, o programa exibe no console a mensagem:

O Herói tem de saldo de {saldoVitorias} está no nível de {nivel}

Exemplo:

O Herói tem de saldo de 30 está no nível de Prata
